FAIRification of Genomic Tracks

ELIXIR Implementation Study 2018-2019

FAIRtracks is a JSON Schema defining a minimal standard for genomic track metadata. FAIRtracks is supported by the TrackHub registry, the TrackFind metadata search and curation engine and the downstream track analysis tools GSuite HyperBrowser and EPICO.
